# User Guide

## Untuk Pengunjung

### Melihat Katalog

1. Buka halaman beranda.
2. Telusuri daftar sound effect terbaru.
3. Gunakan kategori atau pencarian untuk mempersempit hasil.

### Membuka Detail Sound Effect

1. Klik salah satu sound effect.
2. Lihat deskripsi, kategori, metadata, dan rekomendasi suara terkait.

### Mengunduh Audio

1. Buka halaman detail sound effect.
2. Klik tombol unduh.
3. File audio akan diunduh ke perangkat.

### Menjelajah Sound Pack

1. Buka menu sound pack.
2. Gunakan filter kategori, negara, atau pencarian.
3. Buka sound pack untuk melihat isi audio di dalamnya.

## Untuk Pengguna yang Mengirim Pesan

### Form Kontak

1. Buka halaman kontak.
2. Isi nama, email, subjek, dan pesan.
3. Kirim formulir.

### Newsletter

1. Masukkan email pada form subscribe.
2. Kirim formulir untuk berlangganan pembaruan.

## Untuk Admin

### Login Admin

1. Buka halaman login.
2. Masuk menggunakan akun yang memiliki akses admin.
3. Arahkan ke dashboard admin.

### Mengelola Kategori

1. Buka menu kategori.
2. Tambahkan kategori baru.
3. Ubah atau hapus kategori bila diperlukan.

### Mengelola Sound Effect

1. Buka menu sound effect.
2. Tambahkan judul, deskripsi, keyword, kategori, dan file audio.
3. Simpan data.
4. Edit atau hapus item jika diperlukan.

### Mengunggah Media

1. Gunakan form unggah gambar atau audio.
2. Pilih file sesuai format yang didukung.
3. Simpan dan gunakan URL hasil unggahan.

## Tips Penggunaan

- gunakan judul yang jelas dan deskripsi yang deskriptif
- pastikan file audio sudah sesuai format dan ukuran yang disyaratkan
- isi kategori agar hasil pencarian dan filter lebih akurat
- perbarui metadata SEO untuk halaman penting

## Masalah Umum

### File Tidak Bisa Diunduh

- pastikan file audio tersedia di folder publik
- cek apakah path file di database sudah benar

### Hasil Pencarian Kosong

- gunakan kata kunci yang lebih spesifik
- pastikan konten memang sudah terdaftar dalam sistem

### Upload Gagal

- periksa format file
- pastikan ukuran file tidak melebihi batas
- cek izin akses folder publik

### Halaman Admin Tidak Terbuka

- pastikan sudah login
- pastikan akun memiliki akses admin
